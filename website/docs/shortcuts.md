# Keyboard Shortcuts

`adbt` relies heavily on intuitive, Vim-like keyboard navigation. Here is a cheat sheet of all available shortcuts.

## Global
| Key               | Action           |
| ----------------- | ---------------- |
| `q` / `Ctrl+C`    | Quit             |
| `Esc`             | Back / Cancel    |
| `↑` `↓` / `k` `j` | Navigate         |
| `Enter`           | Select / Confirm |

## Dashboard
| Key | Action              |
| --- | ------------------- |
| `d` | Devices             |
| `m` | Performance Monitor |
| `a` | App Manager         |
| `f` | File Explorer       |
| `l` | Logcat              |
| `i` | Device Info         |

## App Manager
| Key     | Action               |
| ------- | -------------------- |
| `/`     | Search               |
| `f`     | Filter (User/System) |
| `s`     | Force Stop           |
| `x`     | Clear Data           |
| `u`     | Uninstall            |
| `Enter` | Launch               |

## File Explorer
| Key         | Action    |
| ----------- | --------- |
| `p`         | Pull File |
| `d`         | Delete    |
| `Backspace` | Go Up     |
